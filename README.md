# Playwright Tasks Tests

Тренировочный проект автоматизации тестирования, покрывающий 24/25 сценариев взаимодействия с Playwright на тренировочном стенде [UI Testing Playground](http://uitestingplayground.com).

## Особенности проекта:

*   **Параллельный запуск**: Запуск тестов в несколько потоков.
*   **Наглядная отчётность**: Автоматическая запись **Playwright Traces** для каждого теста.
*   **Интеграция с Allure**: Все результаты, включая трассировки и логи, автоматически прикрепляются к отчету.
*   **Перезапуск тестов**: При падении теста он запускается повторно.

## Технологический стек:

*   **Фреймворк**: Playwright 1.56.0
*   **Движок запуска тестов**: JUnit 5.10.0
*   **Отчётность**: Allure Report 2.33.0
*   **Сборщик**: Gradle 8.13
*   **JDK**: 17.07.17

## Инструкция по запуску:

### 1. Запуск тестов:
Для запуска  тестов использовать:

`./gradlew clean test`

### 2. Работа с отчетами Allure:
Сгенерировать и открыть отчет в браузере:

`./gradlew allureServe`

### 3. Просмотр трассировок:
Файлы трассировок сохраняются в отчёте `Allure`. Чтобы просмотреть их:
- Открыть `Allure` отчёт;
- Выбрать нужный тест;
- Во вкладке `Overview` раскрыть список `Tear down` - `tear down` - `Playwright Trace` - `Click to download attachment` в формате .zip;
- Открыть сайт `trace.playwright.dev`;
- Перетащить туда скаченный архив в формате .zip.
